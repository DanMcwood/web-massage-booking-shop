**Massage Booking & Shop**

Полнофункциональный Django веб-сайт для онлайн-заказа и бронирования услуг массажа и товаров для массажа.

Проект демонстрирует полный цикл разработки веб-приложения: от архитектуры и бэкенда до адаптивного фронтенда и интеграции системы бронирования с интернет-магазином.

**Описание проекта:**

Massage Booking & Shop — это коммерческий прототип веб-приложения, объединяющий интернет-магазин и систему онлайн-бронирования.

**Цель проекта:**

Предоставить пользователю удобный способ покупки товаров и записи на услуги массажа.

Продемонстрировать навыки full-stack разработки, включая Django, фронтенд, работу с базой данных и админ-панель.

**Ключевые возможности проекта:**

Каталог товаров с карточками, фильтрацией и категориями.

Корзина с сохранением сессий и умным оформлением заказа.

Личный кабинет с регистрацией, подтверждением email, историей заказов и бронирований.

Онлайн-бронирование услуг с выбором даты и времени.

Админ-панель для управления товарами, заказами, бронированиями и экспортом данных в Excel.

**Стэк:**

Backend: Django 5, Python 3.11

Frontend:	HTML, CSS, JavaScript

Database:	SQLite (легко мигрируется на PostgreSQL)

Дополнительно:	Django Import-Export, сессии для корзины

**Главные фичи:**

Автозаполнение форм: пользователям не нужно вводить информацию повторно — данные подставляются из профиля.
Адаптивный фронтенд: интерфейс оптимизирован под десктоп и мобильные устройства.
Полная интеграция магазина и бронирования: единый аккаунт и единая база данных для всех операций.
